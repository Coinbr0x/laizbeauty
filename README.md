# LaizBeauty — Professional Beauty Salon & Aesthetics

A premium, interactive, dark-themed landing page designed for a high-end beauty salon. The site showcases custom visual effects, services, and a booking system.

Live Demo: [https://laizbeauty-6f7f7.web.app](https://laizbeauty-6f7f7.web.app)

---

## 🌟 Key Features

*   **Interactive Spotlight Reveal**: A custom cursor-following spotlight effect built on HTML5 Canvas that masks and reveals a transition from natural beauty (`nomakeup.png`) to professional glamour (`withmakeup.png`).
*   **Services Showcase**: A responsive, glassmorphic card grid displaying salon services (makeup, skin therapy, hair styling, brow design) with smooth hover micro-animations.
*   **Bespoke Reservation Form**: A fully integrated booking form with state validation, simulated API submission, and a clean booking success confirmation overlay.
*   **Responsive Architecture**: Built mobile-first with viewport height styling safeguards and fully fluid layouts.

## 🚀 Getting Started

### Prerequisites

*   Node.js (v18 or higher)
*   npm or yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Coinbr0x/laizbeauty.git
    cd laizbeauty
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run locally**:
    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```

5.  **Deploy**:
    ```bash
    firebase deploy --only hosting
    ```
